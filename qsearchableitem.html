    Detailed Description

    Each item has a mandatory uniqueIdentifier and optional metadata such as title, contentDescription, keywords, and more. Items can be grouped by domainIdentifier so they can be removed as a batch with QSearchableIndex::removeItemsInDomains().

    QSearchableItem uses implicit sharing, so copies are inexpensive.

    See also QSearchableIndex.
