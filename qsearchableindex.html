    Detailed Description

    The following platforms are supported:

    • macOS: Spotlight
    • Windows: Windows Search
    • Linux (GNOME Shell): GNOME Tracker
    • Linux (KDE Plasma): KRunner

    On unsupported platforms a no-op backend is used and isSupported() returns false.

    QSearchableIndex is a singleton. Use Get() to obtain the instance.

    Example

    QSearchableItem item("contact-42");
item.setTitle("Alice");
item.setContentDescription("alice@example.com");

QSearchableIndex::Get()->indexItems({item});

    See also QSearchableItem.

    Indexes (or updates) the given items. When the operation completes the indexingSucceeded() signal is emitted.

    Emitted after indexItems() completes successfully. count is the number of items that were indexed.
